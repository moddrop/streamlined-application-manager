SamOS

SamOS, is an operating system made for launching Streamlined applications, tense the name. It was created by 'md'.

Background

md began developing SamOS in 2025. Its earliest build had a really noisy feature where when the cursor hovers the titlecard it makes a bell ringing sound.

System Overview

All of the versions of the system includes simple css default colors, such as: blue, red, purple, green.
